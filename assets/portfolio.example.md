# Example Portfolio

## Project 1

**일정**: 2025.01 - 2025.03

**역할:** Backend Developer

**핵심 키워드:** API, 데이터 모델링, 운영 자동화

### 문제

- 반복되는 운영 작업과 수동 검증 비용이 컸습니다.

### 해결

- 간단한 내부 자동화 도구를 만들고 검증 절차를 문서화했습니다.

### 성과

- 반복 작업 시간을 줄이고 팀이 같은 기준으로 검토할 수 있게 했습니다.

### 배운 점

- 기능 구현보다 먼저 문제를 작게 나누고 검증 가능한 흐름을 만드는 것이 중요했습니다.
