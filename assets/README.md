# Assets

공통 자산을 두는 폴더입니다.

여기에는 실제 자기소개서 작성에 반복해서 쓰는 개인 자료를 정리합니다.

- `resume.md`
- `portfolio.md`
- `credentials.md`
- `cv-bullets.md`
- `past-applications/`

## 무엇을 넣는지

- `resume.md`: 이력서와 기본 경력 정보
- `portfolio.md`: 프로젝트 상세, 역할, 성과, 배운 점
- `credentials.md`: 자격증, 수상, 교육, 대외활동 같은 정량 정보
- `cv-bullets.md`: 짧은 경력 bullet 정리
- `past-applications/`: 과거 지원서와 문항별 기록

형식은 `*.example.md`를 참고하면 됩니다.
